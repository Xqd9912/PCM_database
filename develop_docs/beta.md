# beta版功能：Structure Descriptors Visualization

**功能描述**：在材料页进行筛选后，对筛选后展示的材料组进行高维结构特征化，然后做PCA降维，给出全局的分布图。
例如：在页面筛选GeTe得到71个结构，对所有结构使用SOAP进行encode，然后pca降维度（对结构的特征矩阵n*m，n为原子数，m为特征维度，做平均得到单个结构的特征量），得到结构分布MAP，71个结构点。
SOAP方法使用方法：https://singroup.github.io/dscribe/latest/tutorials/descriptors/soap.html

**进阶功能**：MAP做成交互模式，通过鼠标指针选取对应的结构点，匹配到结构对应的条目。需要说明的是，为了匹配到对应的条目，在材料显示页面增加一栏序号用于暂时标记每个结构id，匹配时显示这个结构id。

# 需要讨论的设计细节
结构特征化需要一定的计算量，在设计开始前先确定计算方案。是先对所有结构预先计算特征量然后筛选后索引对应结构，还是筛选之后即时计算特征。
